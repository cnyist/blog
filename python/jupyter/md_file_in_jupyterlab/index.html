jupyter lab 中编辑 .md文件

发表于2020-02-26|更新于2021-08-14|PythonJupyter
|阅读量:

本来以为 jupyter lab 里没办法预览 .md 文件, 还找了好久插件, 结果其实不用插件也可以, 只要打开 .md 文件, 然后右键就有 Show Markdown Preview 选项, 点了之后就可以在右半边实时预览.
好吧这是一篇水文.
